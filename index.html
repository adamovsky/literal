Welcome to Literal

Literal is a high performing lightweight utility library that helps with manipulating literal data structures in JavaScript.

We all write code like this... 

if (myObject 
  && myObject.items 
  && myObject.items.item 
  && myObject.items.item.property) {
  // ...
}

Now we can simple tell literal which path we want to make sure exists, and it will check all nodes for us. 

if (literal(myObject).check("items.item.property")) {
  // ... 
}

A slightly more complex example

We can also very easily swap content around within a given data structure. 

if (myObject 
  && myObject.items 
  && myObject.items.item 
  && myObject.items.item.property
  && myObject.someArray
  && myObject.someArray.length > 0) {
  var first = myObject.items.item.property;
  var myArray = myObject.someArray;
  if (myArray[4]
      && myArray[4].hasOwnProperty("map")) {
    var second = myArray[4].map.item;
  }
  if (first && second) {
    myObject.items.item.property = second;
    myArray[4].map.item = first; 
  }
}
          
literal(myObject).swap("items.item.property", "someArray[4].map.item");

Yes, all that code replace with one line.
