Welcome to Literal

Literal is a lightweight and high performance utility library to help with common manipulation operations when dealing with literals in JavaScript.

So what can this library help with ?

if (myObject 
    && myObject.items 
    && myObject.items.item 
    && myObject.items.item.property) {
    // ...
}

if (literal(myObject).check("items.item.property")) {
    // ...
}
